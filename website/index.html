Why Theonix

A distribution built with intent

Every layer — installer, repository, desktop and branding — is tuned so the system is coherent from first boot.

Rolling Arch base

Built on Arch Linux with Pacman. Install once and receive continuous updates — no version jumps, no reinstalls.

KDE Plasma 6 · Wayland

A modern Plasma 6 desktop running on Wayland by default, with the custom Theonix theme and branding throughout.

Custom repository

A dedicated Theonix package repository ships the branding, defaults and tooling that make the system feel complete.

Guided installer

A branded Calamares installer handles partitioning, users and the bootloader — from live USB to installed desktop in minutes.

Fast boot

A lean systemd service set and a streamlined boot path get you to the desktop quickly, with Fastfetch showing off the result.

AI integration Planned

A local, privacy-respecting AI assistant is on the roadmap — on-device help without sending your data to the cloud.
