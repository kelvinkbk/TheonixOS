FAQ

Frequently asked questions

Short, honest answers about what Theonix OS is and how to run it. Can’t find what you need? Ask the community.

General

What is Theonix OS?

A modern Linux distribution based on Arch Linux, with KDE Plasma 6 on Wayland, a rolling release model, a custom Calamares installer and its own package repository.

Is it free and open source?

Yes — free to download and use, released under the GPL.

What is it based on?

Arch Linux, using the Pacman package manager, with Flatpak available for sandboxed apps.

Which desktop does it use?

KDE Plasma 6 on Wayland, with the custom Theonix theme and branding.

Is it a rolling release?

Yes. Install once and keep updating — there are no fixed versions to reinstall over.

Does Theonix have an AI assistant?

Not yet. A local, privacy-respecting AI assistant is Planned for a future release — it is not part of the current version.

Installation

What are the system requirements?

Minimum: dual-core 64-bit CPU, 4 GB RAM, 25 GB storage. Recommended: quad-core CPU, 8 GB RAM, 50 GB SSD, GPU with OpenGL 3.3.

How do I install it?

Download and verify the ISO, write it to a USB stick, boot the live session, and run the Calamares installer. See the documentation.

Can I dual-boot with Windows?

Yes. Back up first, leave free space for Theonix and install alongside your existing OS — GRUB will detect other systems. See the dual-boot notes.

Do I need to disable Secure Boot?

You may need to disable Secure Boot in your firmware to boot the live media. See the Secure Boot section.

Usage & support

How do I update Theonix?

Update system packages with sudo pacman -Syu and Flatpak apps with flatpak update. As a rolling release, updating regularly is recommended.

How do I get help or report a bug?

Ask on the forum or open an issue on GitHub. See the community page for all channels.
