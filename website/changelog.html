1.0 “Genesis”

Stable First release 2026

The first public release of Theonix OS — a coherent, ready-to-use Arch-based desktop with the Theonix branding, a custom installer and its own package repository.

  • ＋Initial public release of Theonix OS.
  • ＋New Theonix branding and theme across the desktop.
  • ＋Custom Calamares installer for a guided setup.
  • ＋Custom Theonix package repository.
  • ＋Fastfetch branding for system information.
  • ✎KDE Plasma 6 desktop environment.
  • ✎Wayland session enabled by default.
  • ✎Improved bootloader configuration.
  • ✗Fixed GRUB installation.

Pre-release development

Internal

Before Genesis, Theonix was built and iterated on privately. This period focused on bringing up the base system on Arch Linux, integrating and refining the Calamares installer, and developing the branding and package repository that would define the distribution. These iterations were not published as versioned releases — they laid the groundwork for the 1.0 launch.

  • •Base bring-up on Arch Linux with Pacman.
  • •Calamares installer integration and tuning.
  • •Branding, theming and repository work.
