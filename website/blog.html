Announcing Theonix OS 1.0 “Genesis”

The first public release of Theonix OS is here. Genesis brings up a modern, rolling Arch Linux base with KDE Plasma 6 running on Wayland, wrapped in a custom Calamares installer and served by our own package repository. Fastfetch and a cohesive custom theme give the system a consistent identity from first boot — and a reworked bootloader install path makes getting there smoother.

This is a foundation to build on, not a finish line. Genesis is free and open source, and we'd love your feedback as we keep refining it.
