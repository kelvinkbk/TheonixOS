Security

Verify your ISO

Always confirm your download before flashing. Comparing the SHA256 checksum proves the ISO downloaded completely and was not altered on its way to you.

01 Get the official checksum

Each build publishes a SHA256SUMS file next to the ISO. Download it from the official route:

curl -LO https://huggingface.co/datasets/kelvinkbk/TheonixOS-1.0/resolve/main/SHA256SUMS

Only trust the checksum from the official TheonixOS Hugging Face repository — not from a third-party page.

02 Compute the checksum

Run the command for your operating system in the folder containing the ISO.

Linux
sha256sum TheonixOS-1.0.iso

Or verify against the file automatically: sha256sum -c SHA256SUMS

macOS
shasum -a 256 TheonixOS-1.0.iso
Windows · PowerShell
Get-FileHash .\TheonixOS-1.0.iso -Algorithm SHA256

Command Prompt: certutil -hashfile TheonixOS-1.0.iso SHA256

03 Compare the values

Match the hash your machine produced against the value in SHA256SUMS.

They match

Your ISO is intact. You’re safe to flash it to a USB drive and boot.

They differ

Do not use the file. Delete it and download again from the download page if it keeps failing.

04 Signature verification Optional

When a detached GPG signature is published alongside a release, you can also verify authenticity with the Theonix signing key:

gpg --verify SHA256SUMS.sig SHA256SUMS

The checksum confirms integrity; a signature additionally confirms the file came from the Theonix maintainers.
